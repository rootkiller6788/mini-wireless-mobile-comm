# Course Tree - mini-beamforming-massive-mimo

## Prerequisites
1. Linear Algebra: vector/matrix operations, eigenvalues, SVD
2. Signal Processing: complex baseband representation, Fourier transform
3. Probability: complex Gaussian distribution, covariance matrices
4. Wireless Communications: SNR, channel models, capacity basics

## Postrequisites (modules that depend on this)
1. mini-5g-nr-phy: Beam management, CSI reporting
2. mini-channel-model: 3GPP channel models
3. mini-wireless-security: Physical layer security (MIMO AN)
4. mini-cellular-network: Multi-cell massive MIMO deployment
